# Walkr

Walkr is a web application that provides AI-assisted walking route recommendations using the Groq API. The application allows users to describe where they want to walk or what kind of route they are looking for, and returns route suggestions generated by a large language model. The frontend provides a simple interface for interacting with the assistant, while the backend securely handles communication with the Groq API.

## Features

- AI-generated walking route recommendations
- Express backend that proxies requests to the Groq API
- Simple and responsive web interface
- Environment variable support for API credentials
- Separation of frontend and backend components

## Installation

Clone the repository.

```bash
git clone https://github.com/jibrix01/walkr.git
cd walkr
```

Install dependencies.

```bash
npm install
```

Create a `.env` file in the project root.

```env
GROQ_API_KEY=your_groq_api_key
PORT=3000
```

Start the development server.

```bash
npm start
```

The application will be available at:

```
http://localhost:3000
```

## Environment Variables

The following environment variables are required.

| Variable | Description |
|----------|-------------|
| `GROQ_API_KEY` | Your Groq API key |
| `PORT` | Port used by the Express server |
