Wharf is a web application that provides AI-assisted walking route recommendations using the Groq API. The application allows users to describe where they want to walk or what kind of route they are looking for, and returns route suggestions generated by a large language model. The frontend provides a simple interface for interacting with the assistant, while the backend securely handles communication with the Groq API.

Created for HACKENG'G 2026. 🥇 1st Runner Up.

## Features

### Personalized Walking Recommendations
Users can describe the kind of walk they want, whether it's a relaxing stroll, a fitness walk, or a route with scenic views. Walkr generates recommendations tailored to the user's preferences instead of providing generic directions.

### Context-Aware Suggestions
The AI considers the details provided in the user's prompt to produce recommendations that match their intended walking experience. Users can specify factors such as the purpose of the walk, preferred environment, or desired atmosphere.

### Route Descriptions
Each recommendation includes a description of the suggested route, highlighting what users can expect during their walk. The responses aim to provide enough context for users to choose a route that fits their preferences.

### Responsive Design
The application was built in mind with mobile interfaces, allowing users to use the app on the go.

## Installation

Clone the repository.

```bash
git clone https://github.com/jibrix01/walkr.git
cd walkr
```

Install dependencies.

```bash
npm install
```

Create a `.env` file in the project root.

```env
GROQ_API_KEY=your_groq_api_key
PORT=3000
```

Start the development server.

```bash
npm start
```

The application will be available at:

```
http://localhost:3000
```

## Environment Variables

The following environment variables are required.

| Variable | Description |
|----------|-------------|
| `GROQ_API_KEY` | Your Groq API key |
| `PORT` | Port used by the Express server |
